\section{Introduction}
Database configuration tuning is expensive and expert-driven.
We study how \pbt can automate this process for PostgreSQL \cite{jaderberg2017pbt}.
We present \sysname, a Population-Based Training system for automatic database parameter tuning.
Our contributions are summarized as follows.
\begin{itemize}
\item We adapt Population-Based Training to parallel PostgreSQL configuration evaluation.
\item We introduce workload-aware scoring and normalization for diverse benchmarks.
\item We provide an end-to-end system and evaluation pipeline for reproducible tuning experiments.
\end{itemize}
