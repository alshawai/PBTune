## Scope

These requirements cover the PVLDB paper workspace and contributor
setup guidance.

## Requirements

- WHEN contributors start writing, THE SYSTEM SHALL provide a PVLDB
  LaTeX skeleton under `papers/pbtune-pvldb`.
- WHEN the PVLDB template files are missing, THE SYSTEM SHALL document
  the required files and where to place them.
- WHEN authors edit the paper, THE SYSTEM SHALL keep each major section
  in its own file under `sections/`.
- WHEN contributors set up their environment, THE SYSTEM SHALL provide
  a guide listing required packages, tools, and recommended VS Code
  extensions.
- THE SYSTEM SHALL define macros for the system name and recurring terms.
